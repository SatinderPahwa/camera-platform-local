Add New Camera - Step 1 of 6
Step 1
Step 1: Flash Camera with Unlocked Firmware
Before you begin: You need to flash the camera with unlocked firmware using AmbaUSB tool on Windows.
Download Required Tools:
  • AmbaUSB Flashing Tool - Get from Google Drive or download from web
  • Unlocked Firmware File - Get from Google Drive
  • FileZilla (FTP Client) - Download from filezilla-project.org (needed for uploading config files in later steps)
  • USB Cable - Note: Only certain types of USB cables work
  • Pin or paperclip - For pressing reset button
Flashing Steps:
  1. Install AmbaUSB on a Windows computer
  2. Open AmbaUSB/bin/ambausb.exe
  3. Choose Board Config: Select S2E.DDR3.EMMC from dropdown
  4. Select Firmware: Browse and select the unlocked firmware file
  5. Connect Camera:
    • Keep pressing the reset button with a pin
    • While holding reset, connect USB cable to camera
    • You should see: "S2E(A9S) is Ready"
  6. Flash Firmware: Click the blue arrow to download firmware
  7. Wait until progress reaches 100%
Next Step: Once firmware flashing reaches 100%, disconnect the camera and proceed to Step 2 for initialization via serial connection.
Important: Unlocked firmware is required for root access, FTP transfers, database modifications, and full camera control via MQTT. You cannot telnet to the camera yet - WiFi must be configured first in Step 2.
Step 2: Camera Initialization
Connect via Serial: Use USB cable and terminal to initialize camera settings and get Camera ID.
Serial Connection Setup:
  1. Connect camera to Mac using USB cable
  2. Find device: In terminal, run: 
    ls /dev/tty.*
    Look for device starting with usb...
  3. Connect to camera: 
    screen /dev/tty.usb... 115200
    Replace /dev/tty.usb... with actual device name
  4. Login: When prompted, enter: 
    User: root
Password: (press Enter - no password initially)
Camera Initialization Commands:
  1. Set root password: 
    passwd root
    Enter password twice (we typically use: admin123)
  2. Configure WiFi: 
    nmcli dev wifi connect <wifi-name> password <wifi-password>
  3. Run shipment command: 
    shipment
    This configures production settings
  4. Get Camera ID: 
    cd /cali/certs
ls *.crt
    Note down the Camera ID from certificate filename (32 hex characters before .crt)
  5. Get IP address: 
    ifconfig
    Note the camera's IP address on your network
  6. Reboot camera: 
    reboot
Important Information to Save:
  • Camera ID (32 hex characters)
  • Camera IP address
  • Root password (admin123)

Troubleshooting

If camera doesn't connect:

  • Check camera can reach server IP via telnet
  • Verify certificates uploaded correctly: wc -c /root/certs/mqttCA.crt should show 1187
  • Check camera logs: tail -f /var/log/messages | grep -i https
